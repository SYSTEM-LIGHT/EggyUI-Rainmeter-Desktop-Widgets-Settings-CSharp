>    - 付费销售或分发

>    - 广告盈利
>    - 商业服务集成
> 4. **素材来源声明：** 本项目开发过程中：
>    - ❌ **从未使用《蛋仔派对》游戏解包/拆包素材**
>    - ✅ 所有视觉元素均为通过合法途径获取（如游戏内截图、官方宣传素材等）
>    - ✅ 所有相似元素均为根据游戏UI风格进行的原创重制
